# 登录盒子

> 为方便开发调试，特推出登录盒子模块

## 组件名称 [loginBox] 可根据引入进行自定义修改

> 该组件依赖于 antv [form、input、message]

## 参数

| 参数名称 | 类型    | 默认值 | 作用                       |
| -------- | ------- | ------ | -------------------------- |
| password | String  |        | 设置默认密码               |
| username | String  |        | 设置默认账号               |
| autoMsg  | Boolean |        | 是否自动提示并拦截错误信息 |

## 方法

| 方法名称 | 类型     | 返回参数                         | 作用         |
| -------- | -------- | -------------------------------- | ------------ |
| onSubmit | Function | (e: EventBySubmitParams) => void | 用于提交表单 |
